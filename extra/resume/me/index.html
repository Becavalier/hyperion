开源项目

cgc C / Assembly

https://github.com/Becavalier/cgc

cgc 是一个用于 C 语言的实验性垃圾收集器，GC 算法基于 Mark-and-Sweep。

cpplax C++

https://github.com/Becavalier/cpplax

cpplax 是 Lax 语言的 C++ 实现，内部包含一个 “tree-walking” 解释器、一个字节码编译器和对应的虚拟机。

twvm C++

https://github.com/Becavalier/TWVM

twvm 是一个实验性的 WebAssembly 虚拟机，支持完整的 Wasm MVP 标准特性。

brainfuck-jit-interpreter C++

https://github.com/Becavalier/brainfuck-jit-interpreter

brainfuck-jit-interpreter 是一个 Brainfuck 语言解释器，它自带一个手写的简单 X86-64 JIT 编译器。

(POC) Zero C++

https://github.com/Becavalier/Zero

Zero 是一个基于 WebAssembly 进行 JavaScript 代码加密的实验性 POC。通过将 ASCII 明文代码转译为 VM 内部依赖的 ByteCode，再到可见字符的 Base64，加壳过程大大降低了前端明文代码的可读性，同时也提高了反编译的难度。

SHDB Shell

https://github.com/Becavalier/SHDB

SHDB 是一个专门为 Shell 脚本提供数据存储功能的 NoSQL 数据库，类似 GDBM。它的配置和使用十分简单，空闲时不占用内存和 CPU 资源，数据库资源完全静态化地保存在本地。

Zoomage.js JavaScript

https://github.com/Becavalier/Zoomage.js

Zoomage.js 是一个基于 HTML5 开发的图片手势缩放库。基于它我们可以在 Web 页面上使用类似原生 Gallery 所提供的、可以对图片进行基于手势的拖拽、旋转以及缩放等变换特效。
